# TradingBox

[![Live Demo](https://img.shields.io/badge/Live%20Demo-線上體驗-blue?style=for-the-badge&logo=vercel)](https://trading-box.vercel.app/)

現代化終端風格 (Terminal Style) 預測市場交易儀表板，深度整合 Polymarket 數據與多維度分析工具。

## 主要核心功能

### 1. 雲端模擬持倉 (Mock Portfolio)

- **即時同步**：整合 Supabase 資料庫，實現跨裝置持倉數據即時同步。
- **盈虧追蹤**：自動根據最新市場報價計算持倉價值、盈虧金額與百分比。
- **客製選單**：重新設計的代碼選單支援搜尋、熱門幣種分類（Crypto/Stock）與品牌圖示。
- **隱私安全**：採用 Row Level Security (RLS) 確保使用者持倉數據完全隔離。

### 2. 討論社群與新聞分享 (Hub)

- **即時交流**：內建全功能討論區，支援與全球投資者交流。
- **新聞分享**：支援將重大新聞一鍵分享至討論區並自動生成預覽卡片。
- **伸縮式設計**：分享的卡片在手機端可自由收疊，點擊展開即可閱讀原文。
- **雙向翻譯**：分享的新聞同樣支援一鍵翻譯，消除跨語言資訊差。

### 3. 智慧價格提醒 (Alerts)

- **自動偵測**：背景監控持倉標的波動，自動發布 ±5% / ±10% 的變動提示。
- **重大異動警告**：當標的波動達 ±20% 時，系統會自動標記 `🚨 [重大異動]` 並發出顯著警報。
- **靜音模式**：優化通知體驗，非必要的登入/登出訊息全面改為靜音，僅保留核心交易提醒。

### 4. 即時行情與異動監控 (Movers)

- **多維度分析**：結合新聞、Twitter 消息與實時價量，提供全方位的異動分析。
- **動態 UI**：前三名排行具備呼吸燈效果，結合 Sparklines 顯示 24 小时機率變化。
- **智能翻譯**：整合 MyMemory API，提供最快速的新聞中英對照體驗。

### 5. 極致響應式設計 (RWD)

- **Mobile-First**：針對手機螢幕寬度量身打造的垂直佈局與側欄開關。
- **優化後的登入頁**：具備視覺衝擊力的雙欄式設計（Web 端）與核心價值動態展示。

## 🚀 快速開始 & 可用指令

### 安裝相依套件

請確保您的環境已有 Node.js (建議 v18 以上)。

```bash
npm install
```

### 啟動本地開發伺服器

```bash
npm run dev
```

### 編譯與打包生產版本

```bash
npm run build
```

### 健壯性自動化檢查 (Code Quality Check)

這是一支我們自製的進階檢查指令，它會自動掃描專案內的多餘程式碼。
使用了 **Knip** 來捕捉未被使用到的 `import`、`export` 與未呼叫的套件；並使用 **jscpd** 來偵測重複的代碼片段。

```bash
npm run check
```

### 本地預覽打包結果

```bash
npm run preview
```
